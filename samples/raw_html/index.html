JoCo Looks Back

By Jonathan Coulton

JoCo Looks Back is the first compilation album by Jonathan Coulton. The album contains previously released studio recordings from Coulton's first seven studio releases.

While preparing this CD for release, Jonathan discovered that all previous releases of Code Monkey had been accidentally exported in mono. This album features the first stereo mix of the song. All other songs on the CD are the same as on previous releases.

Though not yet linked from JonathanCoulton.com, the CD is available for purchase from Amazon.
